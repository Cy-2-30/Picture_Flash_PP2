Memory Game

Card Game

The original game is to be played as a round game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards. This game is created from the same theory.

The game can be played with any number of players or as a solitaire or patience game. It is a particularly good game for young children, though adults may find it challenging and stimulating as well. The scheme is often used in quiz shows in fact, several game shows have used its name in their titles and can be employed as an educational game.

Some of the names the game is know as including Memory, Matching Pairs, Match Match, Match Up, Pelmanism, Pexeso or simply Pairs.

Game Rules

  • The game will start with 4x4 cards to which the player will be given a number of moves to turn the cards and match them.
  • The player would have to turn all the cards within the given moves, if fails to do so it will be game over when moves reach reach 0.
  • The player can choose to quit the game at any point.
  • However, should the player turn all the cards within the allocated moves, any remaining moves will be counted as 1 point each towards the score.
  • For each pair the player manages to match, 1 point will be allocated to the player.
  • For each successful completed level the player would be lead to the next game at the end of each completed round.
  • The game difficulty will increase with progress.
  • In the beginning the player has an option to play alone, or with another player or with the computer.

Tips On How To Play

The game theme revolves around "Back To School" them. If the players bares this in mind, it would be easy to remember the pictures flashed.

The player gets 3 hints throughout the whole game. The hints will flash the cards again to the player, however it should be used wisely.

Each picture would have a different background color and this could be used as a hint.

The player can try to turn the cards as quick as possible, while the memory is still fresh. The longer you wait, the more you might forget.
